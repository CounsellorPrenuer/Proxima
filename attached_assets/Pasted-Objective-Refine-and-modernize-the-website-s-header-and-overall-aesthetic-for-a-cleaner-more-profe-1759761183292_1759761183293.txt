Objective: Refine and modernize the website's header and overall aesthetic for a cleaner, more professional, and visually appealing look. The current design feels cluttered and bland; the goal is to introduce minimal but impactful changes to improve visual hierarchy and user experience.

Component to Modify: Header.jsx (or equivalent component) and global CSS file (styles.css or globals.css).

Specific Changes Required:

1. Header & Logo Modifications:

Merge Logo and Home Link:

Remove the separate "Home" text link from the navigation bar.

Make the entire logo element a clickable link that navigates to the homepage (/).

Increase Logo Prominence:

Increase the height of the logo to 50px to make it a more prominent visual anchor. Ensure its aspect ratio is maintained.

Increase Header Size & Spacing:

Increase the header's vertical padding to 1.5rem on the top and bottom to give all elements more breathing room.

Add a subtle box-shadow to the header to create a sense of depth and lift it off the page. Example: box-shadow: 0 2px 8px rgba(0, 0, 0, 0.06);.

2. Navigation Bar De-cluttering:

Reduce Clutter: The navigation bar feels too crowded. To simplify it, group related items.

Create a single navigation item called "About" with a dropdown menu.

Move the existing "Why PROXIMA" and "About Ashu Manchanda" links into this "About" dropdown.

Improve Spacing:

Increase the gap or margin between the remaining top-level navigation links (e.g., About, Services, Mentoria Partnership, etc.) to at least 2rem to prevent them from looking cramped.

3. Aesthetic and Interactive Enhancements:

Link Hover Effects:

Apply a smooth, non-intrusive hover effect to all navigation links (including dropdown items). On hover, the text color should transition to a primary brand color (e.g., the blue from the CTA button).

Add a transition property to make the color change smooth. Example: transition: color 0.3s ease;.

CTA Button Hierarchy:

The two CTA buttons ("Book a Consultation" and "Explore Services") are good but can be refined.

Keep "Explore Services" as the primary, solid-filled button.

For the secondary "Book a Consultation" button, add a subtle border color on hover to make its interactivity more obvious.

4. Overall Website Polish (Global CSS):

Improve Visual Consistency: To make the entire website less bland, apply the same font and interactive principles from the header to other clickable elements like buttons and links throughout the site for a cohesive user experience.

Enhance Section Spacing: Ensure there is generous and consistent vertical padding (e.g., 4rem or 5rem) between each major section of the homepage (Hero, About, Services, etc.) to create a clean, scannable layout.
